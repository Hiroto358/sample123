# 🌱 初めてのプロジェクト

このプロジェクトは、**Vibe Coding**を使って **v0** と **Supabase** で開発を体験するために作成しました。  
AIと対話しながらアプリを形にしていく、**ノーコード×AI開発の第一歩**です。

---

## 🧩 使用技術
- **v0.dev**：UI構築ツール  
- **Supabase**：データベースと認証  
- **OpenAI API**：AI連携  
- **JavaScript / TypeScript**：アプリの動作制御  
- **GitHub**：ソース管理・公開
